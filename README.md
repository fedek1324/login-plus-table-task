# IT Guru Task

Web-application with authorization screen and products page.

**Demo:** <https://login-plus-table-task.vercel.app/>

## Tech Stack

- React 18+ with TypeScript
- Zustand (state management)
- AG Grid (products table)
- Vite (build tool)
- React Hook Form (form validation)

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Production build
npm run build

# Lint
npm run lint
```

## API

- Products: https://dummyjson.com/docs/products
- Auth: https://dummyjson.com/docs/auth
